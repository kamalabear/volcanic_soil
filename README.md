# Volcanic Soil

Adds a fertile, animated volcanic soil block that serves as a unified output for two different soil-generation systems: cooking compressed blocks and melting stone in a crucible. Volcanic Soil is shovel-mined and spreads like dirt, making it useful for building fertile terrain or decorating landscapes.

Both natural and tilled volcanic soil can accelerate sapling growth, while the tilled form also accelerates crop growth and tracks fertility decay over harvest cycles.

The node can be obtained through two recipes: smelting compressed cobble or processing stone in a Lava Crucible. Both paths converge on a single node to reduce inventory clutter and encourage progression.

---

## Dependencies

- **Required:** `moreblocks` (for compressed cobble input), `ore_dust` (for crucible integration)
- **Optional:** `lava_crucible` (enables stone-to-soil conversion)
